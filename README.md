# tw

## Description

You can Twitter on CUI.

## Usage

```
tw [tweet] TEXT...
tw reply
tw rt TWEET_ID
tw dm TEXT...
tw del TWEET_ID
tw search QUERY
tw help
```

## Install

To install, use `go get`:

```bash
$ go get -d github.com/upamune/tw
```

If you use HomeBrew,

```bash
brew tap upamune/tw && brew install tw
```
